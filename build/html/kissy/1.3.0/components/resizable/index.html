Resizable¶

这是一个缩放组件.
支持八种方向缩放,可在上, 下, 左, 右以及左上, 左下, 右上, 右下进行拖动缩放, 配置简单, 非常方便用户调用.
拖放 handler 的样式需要调用者自己编写.

参数配置详情¶

resizable.node¶

{String|HTMLElement|KISSY.Node} - 将要进行缩放的节点

resizable.minWidth¶

{Number} - 默认为0, 表示拖动缩放的最小宽度.

resizable.minHeight¶

{Number} - 默认为0, 表示拖动缩放的最小高度.

resizable.maxWidth¶

{Number} - 默认为 Number.MAX_VALUE, 表示拖动缩放的最大宽度.

resizable.maxHeight¶

{Number} - 默认为 Number.MAX_VALUE, 表示拖动缩放的最大高度.

resizable.handlers¶

{Array<String>} - 默认为 [], 表示可拖动元素的哪些位置来进行缩放. 可取下列值: “b”,”t”,”r”,”l”,”tr”,”tl”,”br”,”bl”． 其中, t,b,l,r 分别表示 top,bottom,left,right, 加上组合共八种取值, 可在上, 下, 左, 右以及左上, 左下, 右上, 右下进行拖动.

resizable.prefixCls¶

{String} - 默认为 sx-, 避免ISV无法定义ks-前缀的样式, ISV可自定义class前缀.

对应DEMO javascript的代码

var Resizable = KISSY.Resizable; //namespace in KISSY

//new Resizable
var r = new Resizable({
	node:".resizable_box", // selector
	handlers:["b","t","r","l","tr","tl","br","bl"], // handlers
	minHeight:50,
	minWidth:50,
	maxHeight:300,
	maxWidth:300
});


//test start 
r.on("resizeStart", function(){
  console.log('resize --  start');
});

//test end 
r.on("resizeEnd", function(){
  console.log('resize --  end');
});

//test destroy 
$(".resizable_box span").on("click", function(){
	r.destroy();
	console.log('resize --  destroy');
});
